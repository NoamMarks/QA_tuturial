בדיקות API ו-Postman

עמוד הבית

מבוא לבדיקות API

בדיקות API מתמקדות בבדיקת ממשקי ה‑HTTP/REST בין רכיבי המערכת, כולל שידור וקבלה של נתונים בפורמט JSON/XML.

Postman ככלי בדיקה

  • יצירת בקשות GET/POST/PUT/DELETE
  • הגדרת כותרות (Headers) ופרמטרים
  • ניתוח תגובות השרת ובדיקת סטטוס קוד
  • כתיבת סקריפטים ב‑JavaScript להצמדות ניסוי (Pre-request & Tests)

תרגול ושאלות

שאלה: כיצד תבדוק שבקשת GET מחזירה את השדות הנכונים?

תשובה: ב‑Postman מוסיפים טסט JavaScript שבודק שה‑response JSON מכיל את המאפיינים הצפויים.
